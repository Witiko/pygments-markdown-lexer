“Pygments Markdown Lexer”
=========================

A `Markdown <https://daringfireball.net/projects/markdown/syntax>`_
lexer for `Pygments <http://pygments.org/>`_ to highlight *Markdown* code
snippets.

    Enables _Pygments_ to handle
